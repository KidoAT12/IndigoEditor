## Getting Started

Now that you have installed Indigo Park (v1.1 and/or v1.02), and have downloaded IndigoEditor and zDEV ue4ss, you are all set for running it!

<h3 align="center"> If you want to use IndigoEditor, then you have to open it from IndigoEditor.exe, not from your Steam library. </h3>

## Controls/Commands

- **Left Click** : to pick up objects and move them around.
- **R** : while holding R, you can rotate the objects with the **Arrow Keys** (movement is disabled when holding R)

****Once you have downloaded zDEV ue4ss, an in-game console will show up by pressing **F10**. From there you can input these commands:****

>- editor_toggle : turns on the editor itself making you able to move objects around.
>- iefly : lets you be able to fly (Space = Up, Ctrl = Down)
>- editor_save (name of map) : Saves your map in your files!  (example: editor_save Test)
>- editor_load (name of map) : Loads map into world, making it playable!  (example: editor_load Test)
